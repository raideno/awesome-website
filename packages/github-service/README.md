# @raideno/github-service

Reusable GitHub client utilities for this monorepo.

## Build

- `npm run build`

## Publish

- `npm publish --access public`

## Features

- Get and update repository files
- Sync YAML list files (with optional README sidecar update)
- Read tags and commits
- Compare commits
- Trigger and inspect workflows
- Update `uses:` action refs inside workflow files
